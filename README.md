
## Обновление: Добавление интерактивной визуализации с помощью Plotly

### Цель обновления
Расширить возможности системы анализа продаж, добавив интерактивную визуализацию данных, что значительно улучшит пользовательский опыт и глубину анализа.

### Что было добавлено

#### 1. Новый модуль plotly_simple.py (130 строк)

Обоснование выбора Plotly:
- Интерактивность: В отличие от статических matplotlib графиков, Plotly позволяет пользователям взаимодействовать с данными (зум, панорамирование, детализация по наведению)
- Веб-совместимость: Графики сохраняются как HTML файлы, которые можно открыть в любом браузере без дополнительного ПО
- Простота интеграции: Plotly легко интегрируется с pandas и не нарушает существующую архитектуру
- Профессиональный вид: Готовые профессиональные шаблоны и цветовые схемы

Комментарий по архитектуре:
plotly_simple.py - спроектирован как минимальный, самодостаточный модуль. Не зависит от существующей системы визуализации matplotlib. Можно использовать отдельно или вместе с существующими графиками.

#### 2. Обновленный main.py с гибридной визуализацией

Обоснование гибридного подхода:
- Совместимость: Система работает даже если Plotly не установлен (используются только matplotlib графики)
- Постепенное внедрение: Пользователи могут продолжать использовать привычную визуализацию
- Автоматическое определение: Система сама определяет доступность Plotly
- Обратная совместимость: Все существующие функции сохранены

Ключевые улучшения UX в main.py:
1. Автоматическая проверка наличия Plotly при запуске
2. Четкое разделение статических и интерактивных графиков
3. Подсказки пользователю при отсутствии Plotly
4. Единый интерфейс для всех типов визуализации


### Архитектурные решения

#### 1.
Вместо переписывания существующей визуализации:
- Добавлен новый модуль plotly_simple.py
- В main.py добавлена условная проверка наличия Plotly
- Существующие классы DataVisualizer и SalesAnalyzer не изменялись

#### 2.
Сохранение единого стиля:
- interactive_charts/ для Plotly графиков
- static_charts/ для matplotlib графиков
- Единые названия анализов в отчетах

#### 3. 
Гибкая система, которая не ломается при отсутствии Plotly:
try:
    from plotly_simple import SimplePlotlyVisualizer
    HAS_PLOTLY = True
except ImportError:
    HAS_PLOTLY = False  # Система продолжает работать с matplotlib

Интерактивность | Нет | Да (зум, панорамирование, детализация)
Формат вывода | PNG/PDF | HTML (веб-совместимый)

Интеграция | Обязательная | Опциональная
